OCR via API

Bạn có thể tạo OCR qua API. Nếu bạn đã đặt mua các gói OCR trả trước và có các trang OCR trong tài khoản của mình, chúng sẽ được sử dụng theo tỷ lệ 1:1 (1 trang OCR qua API = 1 trang OCR), nếu không, số phút API sẽ được sử dụng theo tỷ lệ 1: 3 (1 trang OCR qua API = 3 phút API). Chi phí của các tùy chọn này gần như giống nhau.
